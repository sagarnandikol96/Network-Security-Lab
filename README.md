# Network Security Lab

## What is this project?
A home lab where I practiced ethical hacking using
Kali Linux attacking a vulnerable VM called Metasploitable2.

## Tools I Used
- Nmap - to scan open ports
- Metasploit - to exploit vulnerabilities  
- Wireshark - to capture network packets

## What I Did
1. Scanned target with Nmap - found 20+ open ports
2. Exploited vsftpd 2.3.4 backdoor - got root shell
3. Exploited Samba vulnerability - got root shell
4. Captured Telnet traffic in Wireshark - saw plain text passwords

## Target Machine
Metasploitable 2 - intentionally vulnerable Linux VM

## Disclaimer
This was done in a completely isolated virtual lab.
No real systems were harmed or targeted.
